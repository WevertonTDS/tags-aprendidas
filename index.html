Elemento (h1)

O elemento h1 define o título mais importante de uma página.
Pode ser usado apenas uma vez por página.
Tem um tamanho grande e é mais destacado.

Elemento (p)

O elemento p define um parágrafo de texto,
que pode ser usado várias vezes por página,
e tem um espaçamento entre as linhas e as margens.

Topo

Elemento (mark)

O elemento mark destaca uma parte do texto com uma cor de fundo.
Ele pode ser usado para chamar a atenção para algo importante.

Topo

Elemento (small)

O elemento small diminui o tamanho da fonte de uma parte do texto.
Pode ser usado para mostrar informações secundárias ou menos relevantes.

Topo

Elemento (i)

O elemento i aplica um estilo itálico a uma parte do texto.
Ele pode ser usado para indicar ênfase, citação, termo estrangeiro ou outro propósito.

Topo

Elemento (u)

O elemento u aplica um estilo sublinhado a uma parte do texto
e pode ser usado para indicar erros, links, referências ou outro propósito.

Topo

Elemento (strong)

O elemento strong aplica um estilo forte a uma parte do texto.
Ele pode ser usado para indicar importância, urgência, seriedade ou outro propósito.

Topo

Elemento (ol)

O elemento ol define uma lista ordenada de itens.
Pode ser usado para mostrar uma sequência, uma hierarquia, uma classificação ou outro propósito.
Os itens são numerados automaticamente.

Topo

Elemento (ul)

E aqui está uma lista não ordenada:

  • Item A
  • Item B
  • Item C

O elemento ul define uma lista não ordenada de itens.
Ele pode ser usado para mostrar uma coleção, um grupo, uma categoria ou outro propósito.
Os itens são marcados com um símbolo.

Elemento (li)

O elemento li define um item de uma lista ordenada ou não ordenada.
Ele deve ser usado dentro de um elemento ol ou ul.

Elemento (a)

Também podemos usar links, como este: DIO.me

O elemento a define um link para outra página ou recurso.
Deve ter um atributo href com o endereço do destino.
Pode ter um texto ou uma imagem como conteúdo.

Topo

Elemento (hr)

O elemento hr define uma linha horizontal que separa seções de uma página.
a TAG não tem conteúdo nem atributos.

Topo

Elemento (sub) e (sup)

Podemos usar subscritos e sobrescritos, como este: H2O e E = mc2

O elemento sub define um texto subscrito, que aparece abaixo da linha de base.
Ele pode ser usado para mostrar fórmulas químicas, notas de rodapé, índices ou outro propósito.

O elemento sup define um texto sobrescrito, que aparece acima da linha de base.
Ele pode ser usado para mostrar expoentes, ordinais, citações ou outro propósito.

Topo

Elemento (blockquote)

Podemos usar citações, como esta:

"Insanidade é fazer sempre as mesmas coisas e
esperar resultados diferentes." - Albert Einstein

O elemento blockquote define uma citação longa de outra fonte.
Ele tem um recuo nas margens e pode ter um atributo cite com o endereço da fonte.

Topo

Elemento (Font)

Podemos usar fontes diferentes, como esta:

Esta é uma fonte Arial azul de tamanho 5

O elemento font define a fonte, a cor e o tamanho de uma parte do texto.
Também podem ser usados os atributos face, color e size com os valores desejados.
essa TAG é considerada obsoleta e deve ser evitado.
